Claim number: 002
Title: Wilson's 240-element octonionic construction is a valid E8 lattice
Status: ESTABLISHED
Date: 2026-04-06

--- STATEMENT ---

The lattice L defined in [Wilson2009] Section 2 as the Z-span of 240 specific
octonions has all the defining properties of the E8 root lattice:

  - Exactly 240 minimal vectors (roots), all of squared norm 2.
  - Rank 8 (spans R^8 as a real vector space).
  - Even integral lattice (all inner products are integers; all norms are even).
  - Unimodular / self-dual: the Gram matrix of a Z-basis has determinant 1.
  - Wilson's specific structural properties Ls ⊆ L and 2L ⊆ Ls hold.

These properties are verified computationally in python_project/tests/test_e8_wilson.py.

--- CONSTRUCTION ---

Wilson defines L as the Z-span of 240 octonions in STANDARD_ALGEBRA (identified in
key claim 001 as the unique octonion algebra common to both papers).

Index convention: Wilson i_inf <-> e_0, Wilson i_k <-> e_{k+1} for k = 0,...,6.
Reference: key claim 001.

Type-1 roots (112):  ±e_a ± e_b  for any distinct a, b in {0,...,7}.
  Count: C(8,2) * 4 = 28 * 4 = 112.
  Coordinate structure: exactly two nonzero entries, each ±1.
  Squared norm: 1 + 1 = 2.

Type-2 roots (128):  (1/2)(±e_0 ± e_1 ± ... ± e_7)  with an ODD number of minus signs.
  Count: 2^8 / 2 = 128 (exactly half of all sign patterns have odd minus count).
  Coordinate structure: all eight entries equal ±1/2.
  Squared norm: 8 * (1/2)^2 = 2.

Total: 112 + 128 = 240.
Reference: [Wilson2009] Section 2 — "112 octonions ±i_t ± i_u" and "128 octonions
(1/2)(±1 ± i_0 ± ... ± i_6) which have an odd number of minus signs".

--- LATTICE MEMBERSHIP CRITERION ---

L is the D8+ lattice (one of the two standard realisations of E8 in R^8):

  L = { x in R^8 : all x_i in Z,      sum(x_i) even  }
    u { x in R^8 : all x_i in Z + 1/2, sum(x_i) odd  }

Consistency check:
  Type-1 root ±e_a ± e_b: integer coords, sum in {0, ±2} (all even). ✓
  Type-2 root (1/2)(sign vector) with k odd minus signs:
    coords in Z+1/2, sum = (8-2k)/2 = 4-k.  Since k is odd, 4-k is odd. ✓

Implemented as is_in_L() in python_project/src/e8_wilson.py.

--- UNIMODULARITY (SELF-DUALITY) ---

A Z-basis must include vectors from BOTH cosets of the D8 sublattice.

The D8 sublattice = { x in R^8 : all x_i in Z, sum(x_i) even } is generated by type-1
roots.  Any basis of 8 independent type-1 roots spans only D8, which has discriminant 4
(index 2 in E8).

A valid Z-basis for E8 = D8+ includes WILSON_S = (1/2)(-e_0+e_1+...+e_7) (type-2)
together with 7 independent type-1 roots.  For the specific basis:

  { WILSON_S, e_0+e_1, e_0-e_1, e_0+e_2, e_0+e_3, e_0+e_4, e_0+e_5, e_0+e_6 }

the matrix B (rows = basis vectors) has det(B) = 1, computed by cofactor expansion
along the e_7 column (which has only one nonzero entry: WILSON_S contributes 1/2).

This was verified numerically: det(Gram matrix) = det(B)^2 = 1.

--- WILSON'S ELEMENT s ---

s = (1/2)(-e_0 + e_1 + e_2 + ... + e_7)  [Wilson writes: s = (1/2)(-1 + i_0 + ... + i_6)]

Properties verified:
  - N(s) = 2 (s is a type-2 root with 1 minus sign = odd).
  - s is in L (is_in_L(s) = True).
  - Ls ⊆ L: for all 240 roots r, the product r*s lies in L.
    (By linearity over Z, this implies Ls ⊆ L for the full lattice.)
  - 2L ⊆ Ls: for all 240 roots r, the product r*s-bar lies in L.
    (By right-alternativity: 2r ∈ Ls iff (2r)*s^{-1} = r*s-bar ∈ L,
     since s^{-1} = s-bar/N(s) = s-bar/2.)

These inclusions are proved analytically in [Wilson2009] Section 2 using the Moufang
laws; the computational verification above provides an independent second-pass check.
