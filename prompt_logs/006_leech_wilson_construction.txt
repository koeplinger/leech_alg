--- CONTEXT ---

Λ is a rank-24 Z-module (lattice) embedded inside L³, where L = E8 = the
integral octonions (the Coxeter–Dickson order).  Because L carries the octonion
product, L³ inherits candidate algebraic structure.  The membership conditions
(1)–(3) characterise exactly which elements of L³ belong to Λ.  The question
of whether an octonion-based multiplication rule on R^24 restricts to a product
that closes on Λ is the central research question of this project.

The minimal-vector shell consists of the 196,560 vectors of minimum norm
(N_std = 8), the analogue of E8 roots for the Leech lattice.  (Λ has no
vectors of norm 2, so it has no root system in the classical sense; the
minimal-vector shell plays the equivalent structural role.)

This prompt established Wilson's membership test and verified all three families
of minimal-shell vectors, providing the reference for evaluating any proposed
product.

--- CONSTRUCTION INVESTIGATED ---

Wilson's Leech lattice [Wilson2009 Section 3]:

    Λ = { (x, y, z) ∈ L³ : conditions 1–3 }

    Condition 1: x, y, z ∈ L
    Condition 2: x+y, x+z, y+z ∈ Ls̄  (= 2B, the Coxeter-Dickson ring × 2)
    Condition 3: x+y+z ∈ Ls

    s   = ½(−e₀+e₁+...+e₇)  ∈ L  (type-2 root, 1 minus sign = odd)
    s̄  = ½(−e₀−e₁−...−e₇)  ∈ R = L̄  (conjugate lattice of L)
    Ls̄ = {λ × s̄ : λ ∈ L}   (index-16 sublattice, det = 256)
    Ls  = {λ × s  : λ ∈ L}   (index-16 sublattice, det = 256)

--- CRITICAL FINDING: TYPE-3 FORMULA ---

Wilson's three families of minimal vectors are:
    Type 1:  (2λ, 0, 0) + cyclic perms
    Type 2:  (λs̄, (λs̄)j, 0) + cyclic perms
    Type 3:  ((λs)j, λk, (λj)k) + cyclic perms

The type-3 formula uses s (not s̄) in the first component.  The initial
implementation used s̄, which produces 184,320 norm-8 triples that do NOT
satisfy Wilson's membership conditions.  This was caught by the is_in_leech
test (test_type3_sample_in_leech failing) and corrected.

Wilson's own paper confirms this: "Dixon's formula is equivalent to
(λs, ±λk, ±(λj)k), that is, omitting the (necessary) multiplication by j
in the first coordinate." — [Wilson2009 p. 2189 footnote]

--- ENGINEERING CHALLENGE: SUBLATTICE Z-BASIS ---

The Ls̄ and Ls Z-bases cannot be found by greedy real-rank selection from the
240 products {λ*s̄} (or {λ*s}), because greedy rank selection over R does not
guarantee integer Z-span coverage.  Instead, the Z-basis for L is found first
(WILSON_S as first element, then greedy type-1 roots, with Gram det = 1 check),
and then the Z-bases for Ls̄ and Ls are obtained as images of the L Z-basis
under right-multiplication.  This is valid because right-multiplication by a
fixed element is Z-linear: if {bᵢ} is a Z-basis for L and λ = ∑nᵢbᵢ ∈ L,
then λ*c = ∑nᵢ(bᵢ*c) for any fixed c.

--- KEY FINDINGS ---

1. Wilson's Leech lattice is verified computationally: all three families of
   minimal vectors satisfy his three membership conditions.

2. The type-3 formula requires s (not s̄) in the first component.  Using s̄
   gives the wrong set of 184,320 vectors.  This is a subtle but critical
   distinction noted explicitly in Wilson's paper.

3. The sublattice Ls̄ = 2B has minimum norm 4: no E8 root (norm 2) is in Ls̄.
   This is the mechanism behind the no-norm-2 proof.

4. The membership test is_in_leech(x, y, z) [Wilson's conditions] is the
   primary reference for evaluating whether any proposed product maps E8
   elements into the Leech lattice.

5. The Leech lattice has 196,560 minimal vectors (norm_std = 8) in three
   families with zero-component count 2, 1, 0 respectively (types 1, 2, 3).
   This pairwise distinctness allows the three families to be identified
   without computing the full membership test.
