--- TRUST PROTOCOL ESTABLISHED ---

Nothing is trusted from any source — peer-reviewed papers, preprints,
Wikipedia, or prior work in this repository. The trust protocol is:
  1. Find a useful property in the literature.
  2. Implement the property in code.
  3. Write a test for the property.
  4. If the test passes, the user reviews it.
  5. Only after user review approval is the property trusted.
