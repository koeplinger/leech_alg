Dixon defines two E8-related sets inside the octonion algebra (STANDARD_ALGEBRA):

Ξ^even (norm² = 1, 240 elements) — [Dixon2010 Section 2, eq. (4)]
  Ξ₀ = {±e_a : a = 0,...,7}                              16 elements
  Ξ₂ = {(±e_a ± e_b ± e_c ± e_d)/2 :                   224 elements
          a,b,c,d distinct,
          e_a(e_b(e_c*e_d)) = ±e_0}
  Ξ^even = Ξ₀ ∪ Ξ₂

A^odd (norm² = 2, 240 elements) — [Dixon2010 Section 6]
  A₁ = {±e_a ± e_b : a,b distinct}                      112 elements
  A₃ = {(1/2)(±e_0...±e_7) : even # of +'s}             128 elements
  A^odd = A₁ ∪ A₃
