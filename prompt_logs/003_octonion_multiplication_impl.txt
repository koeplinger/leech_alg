After reading both papers in full:

Wilson [Wilson2009] uses PL(7) = {inf} u F_7 as his index set.
  Base rule: i_0 * i_1 = i_3.
  Symmetries: t->t+1 and t->2t (mod 7 in {0,...,6}).
  7 triples (Wilson indices): (0,1,3),(1,2,4),(2,3,5),(3,4,6),(4,5,0),(5,6,1),(6,0,2).

Dixon [Dixon2010] uses {0,...,7} as his index set (e_0 = identity).
  Base rule: e_a * e_{a+1} = e_{a+3}, a=1,...,7 (mod 7 in {1,...,7}).
  7 triples (Dixon indices): (1,2,4),(2,3,5),(3,4,6),(4,5,7),(5,6,1),(6,7,2),(7,1,3).

Index mapping: Wilson i_inf <-> Dixon e_0; Wilson i_k <-> Dixon e_{k+1} for k=0,...,6.
Under this mapping the two triple sets are IDENTICAL.
=> ONE multiplication table (STANDARD_ALGEBRA) suffices for both authors' standard product.

Dixon additionally defines an X-product:
  A o_X B := (A * l0) * (l0^{-1} * B),  l0 = (e_0+...+e_7)/2.
  Multiplication table governed by: e_a o e_{a+2} = e_{a+3}, a=1,...,7 (mod 7).
  7 triples: (1,3,4),(2,4,5),(3,5,6),(4,6,7),(5,7,1),(6,1,2),(7,2,3).
=> XPRODUCT_ALGEBRA is a distinct (but isomorphic) octonion algebra.
