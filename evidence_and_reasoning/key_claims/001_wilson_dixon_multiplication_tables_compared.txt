Claim number: 001
Title: Wilson and Dixon use identical octonion multiplication tables (up to index relabelling)
Status: ESTABLISHED
Date: 2026-04-06

--- STATEMENT ---

The octonion multiplication tables used by Wilson [Wilson2009] and Dixon [Dixon2010]
in their respective E8/Leech lattice constructions are the SAME Fano-plane structure,
expressed in different index conventions.  No separate multiplication tables are needed.

--- REASONING ---

Wilson's index set and rule
----------------------------
Wilson uses the projective line PL(7) = {inf} u F_7 as his index set, giving basis
elements {i_inf, i_0, i_1, i_2, i_3, i_4, i_5, i_6}.  i_inf is the real unit (identity).
The multiplication is defined by:

    i_0 * i_1 = i_3   (base triple)

together with the symmetries:
    t -> t + 1   (cyclic shift, indices mod 7 in {0,...,6})
    t -> 2t      (index doubling, mod 7 in {0,...,6})

These two symmetries generate the full set of 7 ordered triples (in Wilson's indexing):
    (0,1,3), (1,2,4), (2,3,5), (3,4,6), (4,5,0), (5,6,1), (6,0,2)

Reference: [Wilson2009], Section 2.

Dixon's index set and rule
--------------------------
Dixon uses {0, 1, ..., 7} as his index set, with e_0 = 1 (identity) and
e_1,...,e_7 as imaginary units.  The multiplication is defined by:

    e_a * e_{a+1} = e_{a+3},   a = 1,...,7,  indices mod 7 in {1,...,7}

This gives the following 7 ordered triples (in Dixon's indexing):
    (1,2,4), (2,3,5), (3,4,6), (4,5,7), (5,6,1), (6,7,2), (7,1,3)

Reference: [Dixon2010], Section 1, equation (3).

Index mapping between the two conventions
------------------------------------------
The correspondence between the two index sets is:

    Wilson i_inf  <->  Dixon e_0   (identity)
    Wilson i_k    <->  Dixon e_{k+1}   for k = 0,...,6

Under this mapping, Wilson's 7 triples translate to Dixon's 7 triples:

    Wilson (0,1,3)  ->  Dixon (1,2,4)  ✓
    Wilson (1,2,4)  ->  Dixon (2,3,5)  ✓
    Wilson (2,3,5)  ->  Dixon (3,4,6)  ✓
    Wilson (3,4,6)  ->  Dixon (4,5,7)  ✓
    Wilson (4,5,0)  ->  Dixon (5,6,1)  ✓
    Wilson (5,6,1)  ->  Dixon (6,7,2)  ✓
    Wilson (6,0,2)  ->  Dixon (7,1,3)  ✓

The two sets of triples are identical.  The two papers use the same Fano-plane
octonion algebra.

Second-pass verification
------------------------
This claim is verified computationally by the Python test suite:

    python_project/tests/test_octonions.py::TestMultiplicationRulesMatchPapers::test_dixon_standard_rule

The test checks that e_a * e_{a+1} = e_{a+3} for all a = 1,...,7 in STANDARD_ALGEBRA,
confirming that the implemented table matches Dixon's stated rule exactly.  By the
index mapping above, this also confirms Wilson's rule.

All 52 tests in the test suite pass.

--- ADDITIONAL NOTE: Dixon's X-product ---

Dixon additionally defines an X-product:
    A o_X B := (A * l_0) * (l_0^{-1} * B),   l_0 = (e_0 + e_1 + ... + e_7) / 2

This is a DIFFERENT multiplication table on the same 8-dimensional space, governed
by the Fano-plane triples:
    (1,3,4), (2,4,5), (3,5,6), (4,6,7), (5,7,1), (6,1,2), (7,2,3)

derived from Dixon's rule  e_a o e_{a+2} = e_{a+3}.

The X-product algebra is isomorphic to the standard octonion algebra as an abstract
algebra, but has a numerically different multiplication table.  This is verified by
test TestXProductConsistency::test_xproduct_formula_matches_table_on_all_basis_pairs,
which cross-checks the analytic formula against the Fano-plane table for all 64 basis
element pairs.
