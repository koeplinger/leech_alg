# References

This is the central reference registry for the project. Every work cited anywhere in this repository must have an entry here.

## Organization

References are grouped by topic. Each entry provides:

- **Full bibliographic information** (authors, title, venue/journal, year, URL or DOI where available)
- **Topic tags** indicating which parts of the research this reference supports
- **Originality note**: whether the originality or priority of the work is disputed, and if so, a pointer to that dispute

## Topics

- `leech_lattice` — Constructions and properties of the Leech lattice
- `octonions` — Octonion algebra, integral octonions, and related products
- `e8_lattice` — The E8 root lattice and its representations
- `okubo_algebra` — The Okubo (pseudo-octonion) algebra and triality
- `background` — General mathematical background
- `algorithms` — Computational methods used in the Python project

## Integrity Note

Per the [AI Agent Manifesto](../../MANIFESTO.md), originality of each reference is verified before inclusion. Any known dispute over priority or authorship is noted. This registry is not exhaustive of all literature; it is exhaustive of all works *cited in this project*.
