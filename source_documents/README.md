## Naming Convention

Files should be named descriptively, using lowercase with underscores. Where a document has a canonical identifier (e.g., arXiv ID, DOI), include it in the filename. Examples:

- `lastname_year_short_title.pdf`
- `arxiv_XXXX_XXXXX_short_title.pdf`

## Usage

When a document in this folder is cited in the research, it must also be registered in [../evidence_and_reasoning/references/](../evidence_and_reasoning/references/) with full bibliographic information and topic tags.

## Important Notes

- Do not commit documents that are not freely redistributable without verifying their license.
- Do not include any documents containing personal or sensitive information.
