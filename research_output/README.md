# Research Output

This folder contains outputs generated during the research: experimental results, analysis summaries, figures, tables, and draft findings.

## Purpose

Research output provides the empirical and computational evidence that accompanies the mathematical reasoning documented in [../evidence_and_reasoning/](../evidence_and_reasoning/).

## Structure

```
research_output/
├── experiments/     # Raw and processed results from Python experiments
├── figures/         # Plots and visualizations
├── summaries/       # Human-readable summaries of findings at key milestones
└── README.md        # This file
```

## Guidelines

- Each experiment output file should reference the Python script or test that produced it.
- Large data files should be excluded from version control (add to `.gitignore`) or stored with appropriate compression.
- Summaries should be dated and versioned so the progression of understanding is traceable.
- No personal or sensitive information is stored here.
