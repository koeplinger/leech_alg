# Mathematical Hypothesis Exploration

This repository documents the exploration of a mathematical hypothesis. It is structured to support rigorous, transparent, and reproducible research, combining formal mathematical reasoning, computational experiments, and careful provenance tracking.

## Repository Structure

| Folder | Purpose |
|---|---|
| [source_documents/](source_documents/) | Primary source materials: research papers, PDFs, and reference texts |
| [prompt_logs/](prompt_logs/) | Chronological log of all AI interaction prompts, enumerated in order |
| [python_project/](python_project/) | Self-contained Python project for sampling, computation, and testing |
| [evidence_and_reasoning/](evidence_and_reasoning/) | Structured documentation of reasoning steps, key claims, and supporting arguments |
| [evidence_and_reasoning/references/](evidence_and_reasoning/references/) | Central reference registry: all cited works, organized by topic |
| [research_output/](research_output/) | Outputs from experiments, analysis results, and draft findings |

## Guiding Principles

All work in this repository is conducted under the [AI Agent Manifesto](MANIFESTO.md). In summary:

- All claims are backed by explicit references.
- Originality disputes for cited works are noted where known.
- No fabricated answers: uncertainty is declared and resolved before proceeding.
- Every AI-assisted answer is independently verified in a second pass.
- Personal and sensitive information is protected throughout.

## Status

This project is in active development. The hypothesis under investigation will be described here once the research framing is established.
